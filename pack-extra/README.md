# pack-extra — Order Bump

Companion pack sold alongside the main ebook. Not required to follow the recipes — adds operational value once a recipe is in production.

| Folder | Content |
|---|---|
| `scripts/` | 10 utility scripts (Excel→Postgres importer, Modbus scanner, OEE PDF report, backup, drift detector, shift merger, log parsers, ...) |
| `dashboards/` | 5 extra Grafana dashboards (OEE daily, energy/machine, Pareto stoppages, quality score, order burndown) |
| `templates/` | Operational audit template, scope-of-work template, financing programmes list |
| `exemplos-financiamento/` | Worked examples of European industrial financing (PT, UK, DE, IT, EU) |

**In development.** Produced during Phase 2.
